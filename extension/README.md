✨ dichthat

Instant translation right where you read.

dichthat is a lightweight Chrome Extension that lets you translate selected text directly on any webpage — no popups, no tab switching.

⸻

⚡ Features
	•	Translate with Alt + W
	•	Right-click → Translate with dichthat
	•	Clean floating tooltip UI
	•	Smart positioning near selected text
	•	Toggle language: VI ↔ EN

⸻

🚀 Usage
	1.	Select any text
	2.	Press Alt + W
	3.	See translation instantly

⸻

🛠️ Tech
	•	Chrome Extension (Manifest V3)
	•	Vanilla JavaScript
	•	Google Translate API

⸻

📦 Install
	1.	Go to chrome://extensions/
	2.	Enable Developer mode
	3.	Click Load unpacked
	4.	Select the extension folder
